# АІС для регулювання роботи продуктового міні-супермаркету «ZLAGODA»
### Керування процесом збуту товарів у продуктовому міні-супермаркеті

*Програмний застосунок призначений для обробки даних, що використовуються в
процесі збуту товарів у продуктовому міні-супермаркеті «ZLAGODA».*
### Система допомагає:
- відслідковувати кількість та ціну продажу товарів;
- вести історію збуту;
- управляти списком усіх типів товарів і їхніми характеристиками;
- контролювати інформацію про працівників та постійних клієнтів (з картками лояльності);
- забезпечувати зручний доступ до даних касирам та менеджерам.

### Типи користувачів
#### 1. Менеджер_ка
   - Контроль товарів, їхньої кількості та цін
   - Перегляд історії продажів
   - Управління працівниками та постійними клієнтами
#### 2. Касир_ка
   - Створення чеків та здійснення продажу товарів

- Шифрування паролів (токенізація)

### Технології та мови програмування
- **Backend:** Go, Java, SQL (PostgreSQL)
- **Frontend:** React
- **Фреймворки/інструменти:** Hibernate, Spring Boot, Swagger, Docker

## Установлення та запуск
1. **Клонувати репозиторій:**
```bash
git clone <URL>
```
3. **Запустити базу даних PostgreSQL (Docker рекомендується)**

4. **Налаштувати конфігураційний файл .env або application.properties для підключення до бази**

5. **Запустити бекенд:**
```bash
mvn spring-boot:run
```
або для Go-проєкту:
```bash
go run main.go
```
6. **Запустити фронтенд:**
```bash
npm install
npm start
```
